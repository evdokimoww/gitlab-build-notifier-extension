# Changelog

Формат вдохновлён [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/); версии по [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.1] — 2026-05-06

### Исправлено

- Уведомления (и связанный звук, префикс заголовка вкладки, favicon и бейдж) больше **не срабатывают при первом заходе** на страницу merge request или пайплайна, если нужный stage уже успел завершиться. Учитываются только **переходы стадии на открытой вкладке**: с «ещё не завершена» (`empty` / `running`) на завершённое состояние (`done`).

### Изменено

- Для отслеживания переходов используется `chrome.storage.session` по ключу вкладка + пайплайн + stage; при закрытии вкладки эти записи удаляются.

## [1.0.0] — начальный релиз

Первая публичная версия: опрос вкладок MR / pipeline / job, уведомления и индикация при завершении выбранного stage CI в GitLab (с учётом fail-fast для `failed`, настроек для `skipped` / `canceled` и whitelist проектов).
